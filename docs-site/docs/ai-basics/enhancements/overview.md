# 增强能力入口说明

这一页保留旧路径，但官方导航已经拆成三条更清晰的主线：

1. [联网增强总览](/docs/ai-basics/online-search/overview)
2. [RAG 与知识库增强总览](/docs/ai-basics/rag/overview)
3. [SPI 网络扩展](/docs/ai-basics/enhancements/spi-http-stack)

这样拆开的原因很简单：

- `SearXNG` 是联网搜索，不是 `RAG`
- `RAG` 应单独讲知识库、分块、embedding、向量召回
- 网络栈扩展又是另一类工程治理能力

如果你是从旧链接进入，建议直接转到新的专题入口阅读。
