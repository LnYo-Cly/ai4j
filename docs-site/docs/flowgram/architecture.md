# Flowgram Architecture

`Flowgram` 这一章的关键结构是：

- `Flowgram.ai` 前端画布 / editor
- starter
- runtime and task APIs
- node execution
- frontend/backend integration

也就是说，这里的 architecture 不是单讲前端库本身，而是讲“前端画布 + AI4J 后端执行层”如何拼起来。
