# 页面已迁移

Provider 配置样例已经迁移到 `Coding Agent` 顶级专题。

迁移原因不是目录调整，而是这部分能力的真实实现位于 `ai4j-cli` / `ai4j-coding`：

- provider/profile 解析链
- workspace 绑定
- session runtime 重绑
- CLI `/provider` / `/model` 命令

因此这个旧页面只保留兼容入口，不再维护第二套正文。

请使用新的正式文档：

1. [配置体系](/docs/coding-agent/configuration)
2. [Provider Profile 与模型切换](/docs/coding-agent/provider-profiles)
