# Why Agent

如果 `Core SDK` 解决的是“模型和能力怎么接”，那么 `Agent` 解决的是“推理循环和编排怎么做”。

## 1. 什么时候该进入 Agent

- 需要多轮推理
- 需要模型自己决定工具调用路径
- 需要 runtime、memory、trace、orchestration

## 2. 它和 Core SDK 的关系

`Agent` 建立在 `Core SDK` 之上，不替代基座。
