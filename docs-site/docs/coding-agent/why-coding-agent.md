# Why Coding Agent

`Coding Agent` 不是“通用 Agent 的另一个名字”，而是面向本地代码仓交付的产品化入口。

## 1. 它额外解决什么

- workspace-aware tools
- session runtime
- host integration
- approvals
- CLI / TUI / ACP

## 2. 它和 Agent 的边界

`Agent` 负责通用 runtime。

`Coding Agent` 负责把这套 runtime 带进本地代码仓任务场景，补上工作区、持续会话、审批和宿主集成。
